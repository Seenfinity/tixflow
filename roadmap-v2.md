## Description

**TixFlow** is an AI-powered event concierge that combines natural language event discovery with cNFT ticketing and community-driven event funding.

Users chat with AI to find events, mint tickets as collectible NFTs, and now—can propose and fund events through our innovative staking pools system.

**The Twist:** Instead of passive ticketing, communities actively shape events by staking tokens to validate demand. When a threshold is reached, the event is funded and executed.

**Stack:** Next.js, TypeScript, Solana (cNFT/Bubblegum), Helius RPC, Phantom Wallet
