## Description Draft

**What is TixFlow?**
TixFlow is an AI-powered event concierge that helps users discover, book, and manage event tickets—minted as cNFTs on Solana blockchain.

**Problem:** Event ticketing is fragmented, expensive, and lacks ownership verification.

**Solution:** AI chat interface + cNFT tickets = seamless booking + verifiable ownership.

**Tech Stack:** Next.js, TypeScript, Solana (cNFT/Bubblegum), Helius RPC, Phantom Wallet
